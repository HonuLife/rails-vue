# README

A starter for working with Rails 7, using Vite as the frontend build tool.

## Features

- ✅ TypeScript support
- ✅ Vue 3
- ✅ Rails 7

## Dev

### Setup
```
bin/setup
```

### Run app:
```
bundle exec rails s
bin/vite dev
```

## Details

### Versions of key dependencies:

- vite@3.0.3
- rails@7.0.3.1
